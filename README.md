# AlgoLens

AST-based static complexity analyzer for C++ using Tree-Sitter.

## Features

- C++ AST generation
- Loop detection
- Recursion detection
- STL algorithm recognition
- Symbolic complexity inference

## Supported Complexities

- O(1)
- O(log n)
- O(n)
- O(n log n)
- O(n²)
- O(n² log n)

## Tech Stack

- Python
- Tree-Sitter
- C++
